### How to play (Control)

-   Mouse Operation : Ship Moving
-   Mouse click : Operates & Shooting start / stop
-   Special mode: If you don't shoot for a while

### Requirement

-   TypeScript, JavaScript
-   AssemblyScript

### How to Build & Run

```sh
$ cd barragewasm
$ npm run asbuild
$ npm start
```

Browse http://localhost:3000
